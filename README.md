## Used Technologies

* React
* TypeScript
* SCSS
* Framer Motion
* Vite
* Swiper

## How to Use

To run this project locally, follow these steps:

### Prerequisites

Make sure you have **Node.js** (v18 or higher) installed on your machine.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/MiladVaghef/Milad-Vaghef-Portfolio-v1.git
   ```

2. Navigate into the project directory:

   ```bash
   cd Milad-Vaghef-Portfolio-v1
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```

This will start the project on `http://localhost:5173` (or another port if 5173 is already in use).

### Production Build

To create an optimized production build:

```bash
npm run build
```

The output will be in the `dist` folder, ready to be deployed.

### Preview Production Build

To preview the production build locally:

```bash
npm run preview
```
